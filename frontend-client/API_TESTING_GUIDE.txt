================================================================================
COMMON HTTP STATUS CODES
================================================================================

200 OK - Request successful
201 Created - Resource created successfully
204 No Content - Request successful, no content to return
400 Bad Request - Invalid request parameters
401 Unauthorized - Missing or invalid authentication token
403 Forbidden - User doesn't have permission to access resource
404 Not Found - Resource not found
409 Conflict - Resource already exists or conflict with current state
422 Unprocessable Entity - Validation errors
500 Internal Server Error - Server error

================================================================================
TROUBLESHOOTING
================================================================================

1. "Invalid API key" - Check REACT_APP_SUPABASE_ANON_KEY is correct
2. "JWT expired" - Token expired, login again to get new token
3. "Row level security policy violation" - User doesn't have permission
4. "Relation does not exist" - Run supabase-setup.sql first
5. "Foreign key violation" - Referenced record doesn't exist
